# 00 Project Scope

## Project

Reusable SUPLEX template repository for adding an agentic control layer to a project without prescribing the underlying computation or publication architecture.

## SUPLEX layer scope

- establish repo-local governance for bounded agentic work
- establish canonical control-memory docs
- establish handoff structure for bounded execution
- establish validation and discrepancy logging structure
- track internal initialization state in `.suplex/init_state.yaml`

## Explicitly out of scope

- computation-layer initialization
- modeling or analysis code scaffolding
- data-layer scaffolding
- notebook scaffolding
- pipeline scaffolding
- publication or deployment scaffolding
- project-specific domain architecture unless separately specified

## Current execution tool or agent context

Generic agentic control-layer template.
