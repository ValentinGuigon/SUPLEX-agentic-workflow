# Execution Report Template

Use this template in `standard` mode when execution reports are stored as dated artifacts in `./.suplex/handoffs/history/`.

## Report Metadata
- task_id:
- executed_by:
- date:
- task_family:
- handoff_reference:

## Files Read

List exact files read.

## Files Created

List exact files created.

## Files Modified

List exact files modified.

## Exact Work Completed

State only what was actually completed.

## Validation Performed

List commands, checks, comparisons, or direct evidence used to validate the pass.

## Acceptance-Criteria Verdict

- [ ] Criterion 1
- [ ] Criterion 2
- [ ] Criterion 3

For any unchecked item, explain why.

## What Was Not Done And Why

List incomplete work, deferred work, or blocked work explicitly.

## Discrepancies Found

List any mismatch between docs, implementation, schema, or task expectations.

## Best-Judgment Assumptions Used

List any material assumption used under user-authorized best judgment.

If none were used, write `none`.

## Recommended Next Task Family

State the likely next bounded task family without claiming authority to choose it.

## Context-Clear Assessment

State whether context can be cleared under the repo rule, and why.
